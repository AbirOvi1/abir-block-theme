# Abir Block Theme

A simple Full Site Editing (FSE) Gutenberg block-based starter theme created for learning and demo purposes.

## What is included
- `style.css` — theme header and basic stylesheet entry
- `theme.json` — global styles & settings (colors, font sizes, layout)
- `templates/index.html` — main template using template-part header/footer
- `parts/header.html` — header template part (site title, logo, navigation)
- `parts/footer.html` — footer template part
- `functions.php` — minimal theme setup and enqueue

## How to use
1. Download the zip file or clone the repository to your local machine or WordPress `wp-content/themes/` folder.
2. Upload the folder to `wp-content/themes/` or install via Appearance → Themes → Add New → Upload Theme.
3. Activate the theme.
4. Go to **Appearance → Editor** (Site Editor) to edit templates, header, footer, and global styles.

## GitHub ready
This folder is structured to be uploaded as a Git repository. You can create a new GitHub repo and push these files.

Example commands:
```bash
git init
git add .
git commit -m "Initial commit — Abir Block Theme"
git branch -M main
git remote add origin https://github.com/<your-username>/abir-block-theme.git
git push -u origin main
```
